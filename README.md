# Permission Doctor

Audit. Diagnose. Repair.

Permission Doctor is a Discord server auditing, security, and setup bot that helps server owners understand permissions, identify security risks, repair misconfigurations, and deploy best-practice server structures.

## Features

### Diagnostics
- /why_can_see
- /why_cant_see
- /role_check
- /channel_check

### Security
- /security_scan
- /server_report

### Repairs
- /fix_role_permissions
- /fix_channel_access
- /sync_category

### Server Setup
- /create_staff_category
- /create_mod_logs
- /create_support_system

### Templates
- /setup community
- /setup gaming
- /setup creator
- /setup business

## Privacy

Permission Doctor does not:
- Store messages
- Track users
- Use analytics
- Sell data
