# Permission Doctor

**Audit. Diagnose. Repair.**

Permission Doctor is a Discord server auditing, security, and setup bot that helps server owners understand permissions, identify security risks, repair misconfigurations, and deploy best-practice server structures.

## Why Permission Doctor?

Managing Discord permissions can become complex as servers grow.

Permission Doctor helps administrators:

* Understand why users can or cannot access channels
* Detect dangerous permissions
* Identify security risks
* Repair permission misconfigurations
* Build structured servers using templates
* Audit roles, channels, and permission overwrites

## Permissions

Permission Doctor may request:

* View Channels
* Send Messages
* Embed Links
* Read Message History
* Manage Roles
* Manage Channels

These permissions are only used when server administrators explicitly invoke repair, setup, or template commands.

Permission Doctor never modifies servers automatically.

## Privacy
