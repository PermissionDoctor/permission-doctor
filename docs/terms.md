# Permission Doctor Support

## About

Permission Doctor helps Discord server owners:

- Diagnose permission issues
- Audit roles and channels
- Detect security risks
- Repair permission problems
- Create best-practice server structures

## Reporting Bugs

2. Include screenshots if possible.

4. Include any error messages.

## Frequently Asked Questions

### Does Permission Doctor store data?

No. Permission Doctor does not store messages or maintain a user database.

### Why can't the bot edit a role?

The bot's role must be higher than the role it is attempting to manage.

### Why can't the bot modify channels?

The bot requires the Manage Channels permission.

### Why did a security scan report a warning?

Warnings indicate potentially risky permissions or configurations that may deserve review.
